# OpenClaw에 오신 것을 환영합니다

## 핵심 개념

- **게이트웨이**: 메시징 플랫폼과 AI 모델 사이의 브릿지
- **에이전트**: AI 기반 대화 인터페이스
- **채널**: 메시지를 주고받는 통신 경로
- **스킬**: 에이전트의 확장 기능
