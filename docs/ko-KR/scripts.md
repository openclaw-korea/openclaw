# 스크립트

`scripts/` 디렉토리는 로컬 워크플로우 및 운영 작업을 위한 헬퍼 스크립트를 포함합니다.
문서나 릴리스 체크리스트에서 참조되지 않는 한, 이러한 스크립트는 **선택 사항**입니다.

## 규칙

- 스크립트는 문서나 릴리스 체크리스트에서 참조되지 않는 한 **선택 사항**입니다.
- CLI 인터페이스가 존재하는 경우 CLI를 우선적으로 사용하세요 (예: 인증 모니터링은 `openclaw models status --check` 사용).
- 스크립트는 호스트별로 다를 수 있으므로 새 머신에서 실행하기 전에 읽어보세요.

## 인증 모니터링 스크립트

## 스크립트 추가 시

- 스크립트는 집중되고 문서화된 상태로 유지하세요.
- 관련 문서에 짧은 항목을 추가하세요 (없다면 새로 작성).
