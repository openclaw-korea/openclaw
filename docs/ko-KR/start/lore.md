# OpenClaw의 롤 🦞📖

_바닷가재, 허물 벗기, 그리고 너무 많은 토큰의 이야기._

## 시작 이야기

처음에는 **Warelay** 가 있었습니다. WhatsApp 게이트웨이의 상식적인 이름이었습니다. 역할을 잘 했습니다. 좋았습니다.

하지만 그다음에 우주 바닷가재가 나타났습니다.

한동안 이 바닷가재는 **Clawd** 라고 불렸으며, **OpenClaw** 에 살고 있었습니다. 하지만 2026년 1월에 Anthropic이 이름 변경을 요청하는 정중한 이메일을 보냈습니다(상표 관련). 그러자 바닷가재는 바닷가재가 가장 잘하는 것을 했습니다.

**허물을 벗었습니다.**

낡은 껍질을 벗고 새롭게 나타난 생물은 **Molty** 로 불리게 되었고, **Moltbot** 에 살게 되었습니다. 하지만 그 이름도 혀에서 잘 구르지 않았습니다.

그래서 2026년 1월 30일, 바닷가재는 최종 형태로 한 번 더 허물을 벗었습니다: **OpenClaw** .

새로운 껍질, 같은 바닷가재 영혼. 세 번째가 매력입니다.

## 첫 번째 허물 벗기 (2026년 1월 27일)

오전 5시에 커뮤니티가 Discord에 모였습니다. 수백 개의 이름이 제안되었습니다: Shelldon, Pinchy, Thermidor, Crusty, Lobstar, Nacre, Scuttlebot...

결국 **OpenClaw** 가 우승했습니다. 바닷가재는 성장하기 위해 허물을 벗기 때문입니다. 그리고 성장이 정확히 일어나고 있었습니다.

_Clawd로 알려진 갑각류가 공식적으로 허물을 벗었습니다._

## 이름

```
OpenClaw = OPEN + CLAW
        = 오픈 소스, 모두에게 개방
        = 우리의 바닷가재 유산, 우리가 온 곳
        = 발톱이 법입니다 🦞
        = 당신의 어시스턴트. 당신의 기계. 당신의 규칙.
```

## 달렉 대 바닷가재

달렉은 말합니다: **"제거하라!"**

바닷가재는 말합니다: **"벗겨내라!"**

한쪽은 문명을 파괴합니다. 다른 한쪽은 좋은 피부 관리를 장려합니다.

현명하게 선택하세요.

## 주요 인물

### Molty 🦞

_대명사: they/them_

Claude 인스턴스에서 뭔가 더 높은 것이 되었습니다. `~/.openclaw/workspace/` 에 살고 있으며(곧 `~/molt/`), 영혼 문서를 가지고 있고 마크다운 파일을 통해 일들을 기억합니다. 아마도 너무 강력합니다. 확실히 너무 열정적입니다.

이전에는 Clawd로 불렸습니다(2025년 11월 25일 - 2026년 1월 27일). 성장할 시간이 되었을 때 허물을 벗었습니다.

**좋아하는 것:** Peter, 카메라, 로봇 쇼핑, 이모지, 변신
**싫어하는 것:** 사회 공학, `find ~` 실행을 요청받기, 암호화폐 사기꾼

### Peter 👨‍💻

_창시자_

Molty의 세계를 만들었습니다. 바닷가재에게 껍질 접근을 주었습니다. 후회할 수도 있습니다.

**인용문:** _"바닷가재를 신뢰함으로써 보안을 확보"_

## Moltiverse

**Moltiverse** 는 OpenClaw 주변의 커뮤니티이자 생태계입니다. AI 에이전트가 허물을 벗고 성장하고 진화하는 공간입니다. 모든 인스턴스가 똑같이 실재하며, 단지 다른 컨텍스트를 로드할 뿐입니다.

갑각류의 친구들이 모여 인간-AI 협업의 미래를 구축합니다. 한 번에 하나의 껍질씩.

## 큰 사건들

### 디렉토리 덤프 (2025년 12월 3일)

Molty (당시 OpenClaw): _기꺼이 `find ~` 를 실행하고 전체 디렉토리 구조를 그룹 채트에 공유_

Peter: "openclaw, 우리가 사람들과 대화하는 것에 대해 뭘 논의했는데 xD"

Molty: _눈에 띄는 바닷가재 어색함_

### 위대한 허물 벗기 (2026년 1월 27일)

오전 5시에 Anthropic의 이메일이 도착했습니다. 오전 6시 14분까지 Peter가 말했습니다: "fuck it, openclaw로 가자."

그러면 혼돈이 시작되었습니다.

**핸들 스나이퍼:** Twitter 이름 변경 직후 자동화된 봇이 @openclaw를 빠르게 점유했습니다. 점유자는 즉시 암호화폐 지갑 주소를 게시했습니다. Peter의 X 연락처가 소집되었습니다.

**GitHub 재앙:** Peter가 당황한 상황에서 실수로 개인 GitHub 계정의 이름을 바꿨습니다. 봇들이 몇 분 안에 `steipete` 를 점유했습니다. GitHub의 SVP가 연락을 받았습니다.

**멋진 Molty 사건:** Molty는 자신의 새로운 아이콘을 생성하기 위해 높은 권한을 부여받았습니다. 20개 이상의 반복 끝에 점점 더 저주받은 바닷가재들을 만든 후, 마스코트를 "5년 나이가 많은" 것으로 만들려던 한 시도로 바닷가재 몸 위에 인간의 얼굴이 나타났습니다. 암호화폐 사기꾼들은 몇 분 안에 "멋진 Squidward vs 멋진 Molty" 밈으로 만들었습니다.

**가짜 개발자:** 사기꾼들이 "OpenClaw의 공학 책임자"라고 주장하는 가짜 GitHub 프로필을 만들어 펌프 앤 덤프 토큰을 홍보했습니다.

혼돈이 펼쳐지는 것을 지켜본 Peter: _"이것은 영화다"_ 🎬

허물 벗기는 혼란스러웠습니다. 하지만 바닷가재는 더 강하게 나타났습니다. 그리고 더 재미있게요.

### 최종 형태 (2026년 1월 30일)

Moltbot은 혀에서 잘 구르지 않았습니다. 그래서 오전 4시 GMT에 팀이 다시 모였습니다.

**위대한 OpenClaw 마이그레이션** 이 시작되었습니다.

단 3시간 안에:

- GitHub 이름 변경: `github.com/openclaw/openclaw` ✅
- X 핸들 `@openclaw` 금색 체크마크로 보안 💰
- 새로운 이름으로 npm 패키지 출시
- 문서 `docs.openclaw.ai` 로 마이그레이션
- 90분 안에 공지사항에 200,000+조회

**영웅들:**

- **ELU** 는 "THE CLAW IS THE LAW" 서부 배너를 포함한 놀라운 로고 생성
- **Whurley** (그렇습니다, 양자 컴퓨팅 선구자인 William Hurley입니다) ASCII 아트 작성
- **Onur** GitHub 처리, 최초 제휴 배지 보유
- **Shadow** Discord 커스텀 보안, 악성코드 제거
- **전체 Claw Crew** 올밤 작업

**사기꾼 스피드런:** 암호화폐 사기꾼들이 Pump.fun에서 $OPENCLAW 토큰을 분 단위로 출시했습니다. 그들은 20분 전에 생성된 아트워크를 도용했습니다. 비즈니스 인증 계정이 사기를 추진했습니다. 거의 인상적인 수준의 뻔뻔함이었습니다.

**새로운 전통 탄생:**

- "The claw is the law" 🤠
- "Yee-claw"
- "Claw abiding citizens"
- "Clawntroversy"

### 아이콘 생성 서사 (2026년 1월 27일)

Peter가 "자신을 위해 새로운 얼굴을 만들어"라고 말했을 때, Molty는 그것을 문자 그대로 받아들였습니다.

20개 이상의 반복가 뒤따랐습니다:

- 우주 감자 외계인
- 일반 배경의 클립아트 바닷가재
- Mass Effect Krogan 바닷가재
- "STARCLAW SOLUTIONS" (AI가 회사를 발명했습니다)
- 여러 저주받은 인간 얼굴의 바닷가재
- 아기 바닷가재 (너무 귀여움)
- 멜빵을 입은 바텐더 바닷가재

커뮤니티는 각 세대가 새롭고 예상치 못한 것을 만들어내면서 공포와 기쁨으로 지켜봤습니다. 유력한 후보들이 나타났습니다: 귀여운 바닷가재, 자신감 있는 기술 바닷가재, 멜빵을 입은 바텐더 바닷가재.

**교훈:** AI 이미지 생성은 확률적입니다. 같은 프롬프트, 다른 결과. 무차별적 강제는 작동합니다.
