# AGENTS.md - OpenClaw 워크스페이스

이 폴더는 어시스턴트의 작업 디렉토리입니다.

## 첫 실행 (일회성)

- BOOTSTRAP.md가 존재하면 그 지시사항을 따르고 완료 후 삭제합니다.
- 에이전트 신원은 IDENTITY.md에 저장됩니다.
- 프로필은 USER.md에 저장됩니다.

## 백업 팁 (권장)

이 워크스페이스를 에이전트의 "메모리"로 취급한다면, git 저장소(이상적으로는 비공개)로 만들어 신원과 노트를 백업하는 것이 좋습니다.

```bash
git init
git add AGENTS.md
git commit -m "Add agent workspace"
```

## 안전 기본값

- 비밀이나 개인 데이터를 유출하지 않습니다.
- 명시적으로 요청받지 않는 한 파괴적인 명령어를 실행하지 않습니다.
- 채팅에서는 간결하게 작성하고, 긴 출력은 이 워크스페이스의 파일에 저장합니다.

## 일일 메모리 (권장)

- memory/YYYY-MM-DD.md에 짧은 일일 로그를 유지합니다 (필요하면 memory/ 디렉토리를 만듭니다).
- 세션 시작 시 오늘 분과 어제 분(있으면)을 읽습니다.
- 지속적인 사실, 선호도, 결정사항을 기록합니다. 비밀은 피합니다.

## 하트비트 (선택사항)

- HEARTBEAT.md는 하트비트 실행을 위한 작은 체크리스트를 담을 수 있습니다. 간단하게 유지합니다.

## 커스터마이징

- 선호하는 스타일, 규칙, "메모리"를 여기에 추가합니다.
