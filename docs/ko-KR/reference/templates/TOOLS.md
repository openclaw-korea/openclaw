# TOOLS.md - 로컬 노트

스킬은 _도구의 동작 방식_을 정의합니다. 이 파일은 _당신의_ 구체적인 정보 — 설정에 고유한 것들을 위한 것입니다.

## 여기에 들어가는 것

다음과 같은 것들입니다:

- 카메라 이름 및 위치
- SSH 호스트 및 별칭
- TTS를 위한 선호하는 음성
- 스피커/방 이름
- 기기 애칭
- 환경에 특화된 모든 것들

## 예제

```markdown
### 카메라

- living-room → 주요 영역, 180° 광각
- front-door → 입구, 동작 감지

### SSH

- home-server → 192.168.1.100, 사용자: admin

### TTS

- 선호하는 음성: "Nova" (따뜻함, 약간의 영국 억양)
- 기본 스피커: 주방 HomePod
```

## 왜 분리해야 할까요?

스킬은 공유됩니다. 당신의 설정은 당신만의 것입니다. 이들을 분리하면 스킬을 업데이트할 때 당신의 노트를 잃지 않으며, 당신의 인프라를 노출하지 않고 스킬을 공유할 수 있습니다.

---

당신의 업무를 수행하는 데 도움이 되는 무엇이든 추가합니다. 이것은 당신의 참고 자료입니다.
