# `openclaw pairing`

DM 페어링 요청을 승인하거나 검사합니다(페어링을 지원하는 채널의 경우).

관련:

- 페어링 플로우: [페어링](/start/pairing)

## 명령어

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```
